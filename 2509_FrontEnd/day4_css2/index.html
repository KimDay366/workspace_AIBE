자기소개

다양한 아이디어를 조합하고, 이를 바탕으로 구체적인 서비스로 구현하는 걸 좋아합니다. 꼼꼼하고 섬세하게 기획하여 다소 느린 편이지만, 그만큼 꾸준히 노력 해 나가는 스타일입니다.

기술 스택

  • Java
  • Spring Boot
  • MySQL
  • AWS
  • GitHub

포트폴리오

여행 추천 프로젝트

사용자의 취향이나 성향을 기반으로 여행지를 추천하는 서비스, DB에 여행지/코스/리뷰 등의 데이터를 넣고 사용자에 따라 필터링 혹은 추천 알고리즘 적용

재능 매칭 프로젝트

사용자 프로필, 카테고리별 재능/스펙/기술등을 등록하여, 다른 사용자와 매칭(키워드 기반 or AI 간단 추천) 기능. 사용자들이 실제 사용 할 채팅 or 예약 시스템 구현

AWS 인프라 프로젝트

클라우드 환경에서 직접 서버/DB/스토리지 올려보고 운영해보는 실습. EC2 인스턴스 띄우기, RDS 세팅, S3에 정적 파일 올리기, 도메인 연결, 로드밸런서/오토스케일링 실습 진행
